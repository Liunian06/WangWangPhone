## 0. 使用说明

这份文档用于持续跟踪“虚拟主屏幕接入 WebView 小组件 + 小组件市场”的 Android MVP 开发。

后续执行规则：
- 每完成一个最小任务，就把对应的 `[ ]` 改成 `[x]`
- 如果方案有变更，先补充文档，再开始改代码
- 所有开发先以 Android 为准，暂不扩展 iOS / Web / Windows
- 优先做 MVP，不做联网市场、不做云同步、不做跨端分享
