# UI 规范：仿微信聊天 (WeChat Clone)

本规范定义了“汪汪机”核心交互模块——虚拟聊天界面的实现细节。目标是像素级复刻微信体验。

## 1. 核心界面 (Chat Interface)

### 1.1 顶部导航栏 (Navigation Bar)
*   **左侧:** 返回箭头 + 消息数量 (如 "微信(12)")。
*   **中间:** 聊天对象昵称 (如 "汪汪")。
*   **右侧:** "..." 更多菜单图标。

### 1.2 消息列表 (Message List)
*   **气泡样式:**
    *   发送方 (右侧): 微信绿 (`#95EC69`), 圆角矩形, 文本黑色。
    *   接收方 (左侧): 微信白 (`#FFFFFF`), 圆角矩形, 文本黑色。
    *   **关键细节:** 气泡尖角 (Tail) 必须精确绘制，与气泡主体无缝衔接。
*   **时间戳:** 自动分组显示 (例如: 超过5分钟间隔显示一次)。
*   **长按菜单:** 黑色半透明背景，包含: 复制, 转发, 收藏, 删除, 多选, 引用, 撤回 (仅限2分钟内)。

### 1.3 底部操作区 (Input Area)
*   **高度:** 自适应内容高度。
*   **组件:**
    *   语音按钮 (左侧): 按下变色，松开发送。
    *   输入框: 自动增长高度。
    *   表情按钮: 切换 Emoji/Sticker 面板。
    *   "+" 号按钮: 切换更多功能面板。

## 2. 功能面板 (Feature Panel)

点击 "+" 号展开，必须包含以下九宫格图标 (参考真实微信图标):
1.  **相册 (Album):** 访问虚拟相册。
2.  **拍摄 (Camera):** 调用虚拟相机。
3.  **视频通话 (Video Call):** 触发 Live2D 视频通话模式。
4.  **位置 (Location):** 发送虚拟位置。
5.  **红包 (Red Packet):** 核心互动功能。
6.  **转账 (Transfer):** 核心互动功能。
7.  **语音输入:** 语音转文字。
8.  **收藏:** 发送收藏内容。
9.  **亲属卡 (Relative Card):** 特有功能，增加好感度。

## 3. 朋友圈 (Moments)

*   **封面:** 支持更换背景图，下拉触发旋转小图标动画。
*   **动态流:**
    *   支持纯文本、图文 (1-9张, 自动排版)、短视频。
    *   **点赞/评论:** 弹出式小黑框交互。
    *   **时间线:** 模拟真实时间流逝。
