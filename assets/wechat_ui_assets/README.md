# WeChat UI Assets Checklist

请参照以下列表从 Figma 导出资源，并放入 `assets/wechat_ui_assets/icons/` 目录。

**推荐格式**: SVG (矢量图，首选) 或 PNG (@3x)。

## 1. 底部导航栏图标 (Tab Bar)

请导出选中 (Selected) 和未选中 (Normal) 两种状态。

| 图标描述 | 建议文件名 (选中 - 绿色) | 建议文件名 (未选中 - 灰色) |
| :--- | :--- | :--- |
| **消息 (气泡)** | `tab_chat_selected.svg` | `tab_chat_normal.svg` |
| **通讯录** | `tab_contacts_selected.svg` | `tab_contacts_normal.svg` |
| **发现 (指南针)** | `tab_discover_selected.svg` | `tab_discover_normal.svg` |
| **我 (人像)** | `tab_me_selected.svg` | `tab_me_normal.svg` |

## 2. 顶部导航栏图标 (Navigation Bar)

| 图标描述 | 建议文件名 | 备注 |
| :--- | :--- | :--- |
| **搜索 (放大镜)** | `nav_search.svg` | 黑色/深灰色 |
| **添加 (圆圈加号)** | `nav_add.svg` | 黑色/深灰色 |
| **返回 (箭头)** | `nav_back.svg` | 黑色/深灰色 |
| **更多 (三个点)** | `nav_more.svg` | 聊天详情页右上角 |

## 3. 聊天界面图标 (Chat Interface)

| 图标描述 | 建议文件名 |
| :--- | :--- |
| **语音 (声波/麦克风)** | `chat_voice.svg` |
| **表情 (笑脸)** | `chat_emoji.svg` |
| **更多功能 (加号)** | `chat_more.svg` |
| **键盘 (切换回文字)** | `chat_keyboard.svg` |

## 4. 通用图标

| 图标描述 | 建议文件名 | 备注 |
| :--- | :--- | :--- |
